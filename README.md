新冠病毒感染人体预测网站
已部署上线，http://82.156.244.145/analyze
输入新冠病毒基因序列，预测出人体可能受到感染的mirna，调控通路
本项目采用 “全栈开发+算法驱动” 的设计理念，自底向上分为数据层、算法层、服务层和表现层：

数据层：
使用 MySQL 构建本地数据库，存储已知的人类 miRNA、宿主 mRNA 及通路数据，并记录用户的分析历史。
开发了特征提取算法，运用滑动窗口技术对长基因组进行切割，提取碱基频率、热力学最小自由能（MFE）、二级结构配对状态等高维特征，并进行张量级归一化。
算法层：
前体预测： 构建了基于 1x1 卷积维度适配与 ResNet（残差网络）的 mirDNN 模型，将核苷酸序列映射为 15 维向量，精准提取序列和结构特征，进行二分类预测。
成熟体提取： 结合 Nussinov 算法与局部堆叠能计算 (miRNA_prediction.py)，模拟 Dicer 酶的切割机制，精准预测成熟 miRNA 及其主导链。
互作与富集： 自研序列互补性打分系统（高/中/低置信度），并运用超几何分布检验，将靶基因映射至生物学通路（Biological Process），从而推断临床症状。
服务层：
基于 Flask 框架搭建后端，采用 JWT (JSON Web Token) 实现用户安全认证。通过面向对象的设计封装 VirusHostMiRNAInteraction 核心类，实现任务的模块化调度。
表现层：
采用 Bootstrap 5 构建响应式交互界面，支持 FASTA 文件拖拽上传。
利用 Chart.js 呈现统计数据，并将复杂的调控网络通过后台降维预计算后，转化为可交互、可下载的 HTML 动态拓扑图。
